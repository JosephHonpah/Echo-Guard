# EchoGuard Combined Authentication Page

I've created a combined login and signup page for EchoGuard that provides a more streamlined user experience.

## Access the Combined Auth Page

**URL**: https://d4t0hj4peur25.cloudfront.net/auth.html

## Features

- **Single Page Authentication**: Login and signup on the same page with tab navigation
- **Modern UI**: Clean, responsive design that works on all devices
- **Cognito Integration**: Full integration with Amazon Cognito for secure authentication
- **Email Verification**: Built-in verification flow for new user signups
- **Automatic Redirection**: Redirects to the main page after successful login

## How to Use

### For New Users:

1. Visit https://d4t0hj4peur25.cloudfront.net/auth.html
2. Click the "Sign Up" tab
3. Enter your email and create a password
4. Submit the form and check your email for a verification code
5. Enter the verification code to complete registration
6. After verification, you'll be redirected to the login tab
7. Log in with your new credentials
