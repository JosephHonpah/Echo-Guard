## Data Flow

1. User uploads audio through React frontend
2. Frontend authenticates with Cognito
3. Audio uploaded to S3 Audio Bucket
4. S3 event triggers StartTranscribe Lambda
5. StartTranscribe Lambda initiates Transcribe job
6. Transcribe results saved to S3 Transcript Bucket
7. S3 event triggers AnalyzeTranscript Lambda
8. AnalyzeTranscript uses Bedrock for compliance analysis
9. Results stored in DynamoDB Audit Log Table
10. Frontend queries API Gateway for results
